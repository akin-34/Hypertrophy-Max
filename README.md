Hypertrophy Max; profesyonel sporcular için tasarlanmış ileri düzey bir performans platformudur. **Güray Training Protokolü** ile **Cerebras Llama 3.1 AI** teknolojisini birleştirerek; hiper-kişiselleştirilmiş beslenme stratejileri, gerçek zamanlı rekabetçi bilgi arenaları ve kapsamlı biyometrik takip modüllerini yüksek performanslı bir "dark-mode" arayüzde sunar.

## ⚡ Temel Zeka Modülleri

### 🧠 AI Beslenme Zekası (Cerebras Core)
Milisaniyeler içinde 7 günlük taktiksel beslenme planları oluşturun. Sistem; metabolik ihtiyaçlarınıza, hedeflerinize ve bütçenize anında adapte olur.
- **Biyometrik Profilleme**: Mevcut formunuz, yaşınız ve aktivite seviyenize göre optimize edilmiş makrolar.
- **Diferansiyel Yollar**: **Naturel** ve **Anabolik** süreçler için özelleştirilmiş farklı algoritmalar.
- **Ekonomik Optimizasyon**: "Öğrenci/Bütçe Dostu" seviyesinden "Elite/Premium" profillere kadar özel menüler.

![AI Beslenme Sonuçları](Seecrennshot/AI%20beslenme%20SOnuç.PNG)

### 🏆 Elite Arena (PvP Bilgi Düellosu)
Bilgi güçtür. Diğer sporcularla gerçek zamanlı, yüksek tansiyonlu bilgi yarışması arenasında kapışın.
- **Gerçek Zamanlı HUD**: Rakibinizin soru ilerlemesini ve skorunu canlı olarak takip edin.
- **Dinamik Soru Setleri**: Anatomi, beslenme ve antrenman bilimi konularında AI tarafından üretilen sorular.

![Elite Arena](Seecrennshot/Yarışma%20online%20anlık.PNG)

### 🤝 Topluluk Merkezi & Mesajlaşma
Elit antrenman partnerleri için özel bir sosyal alan. İletişim kurun ve meydan okuyun.
- **Canlı Sohbet**: Fitness topluluğu ile anlık mesajlaşma.
- **Arena Davetleri**: Entegre "yıldırım işareti" sistemi ile anlık eşleşmeler.

![Topluluk & Sohbet](Seecrennshot/topluluk%20mesajlaşma%20ve%20yıldırım%20işareti%20ile%20arena%20daveti.PNG)

### 📊 Biyometrik Analiz
Vücut ölçüleri, kilo ve beslenme geçmişi için veri odaklı gelişim takibi.
- **Görsel Gelişim**: **Progressive Overload** (Kademeli Aşırı Yükleme) ilkesini korumak için ölçümlerin dinamik takibi.
- **Geçmiş Kayıtlar**: Önceki AI planlarının ve vücut metriklerinin kapsamlı arşivi.

![Analiz & Takip](Seecrennshot/kayıtlık%20ölçüm%20ve%20beslenme.PNG)

## 🛠️ Teknik Mimari

| Katman | Teknoloji |
| :--- | :--- |
| **Mantık (Logic)** | PHP 8.x (Güvenli Backend Proxy) |
| **Zeka (AI)** | Cerebras Inference (Llama 3.1 8B Instruct) |
| **Arayüz (UI)** | TailwindCSS (Modern Glassmorphism) |
| **Veritabanı** | MySQL (İlişkisel Veri Yönetimi) |
| **Çıktı** | HTML2PDF.js (Taktiksel PDF Üretimi) |

## 🚀 Hızlı Başlangıç & Kurulum

1.  **Repoyu Klonlayın**:
    ```bash
    git clone https://github.com/akin-34/Hypertrophy-Max.git
    ```
2.  **Yapılandırma**:
    - `htdocs/config.sample.php` dosyasını `config.php` olarak adlandırın.
    - **Cerebras API Anahtarınızı** ve MySQL bilgilerinizi girin.
3.  **Veritabanı**:
    - Paket içindeki SQL dosyasını veritabanınıza içe aktarın.
4.  **Yayına Alın**:
    - Herhangi bir Apache/PHP web sunucusunda (XAMPP/VDS) çalıştırın.
